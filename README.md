# gba-ES6 - Web GAMEBOY ADVANCE emulator

_Work In Progress_

gba-ES6 is a free web Gameboy Advance Emulator written in ECMAScript 6.

## Build

```
npm install
npm build
```

## Test

```
npm test
```

## Run

```
npm cli <bios file> <rom file>
```
